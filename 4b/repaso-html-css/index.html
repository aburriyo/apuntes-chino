HTML 1. Estructura básica de HTML

Todo archivo HTML tiene la misma estructura base. Es como el esqueleto de la página:

<!-- Esto le dice al navegador que es HTML5 -->
<!DOCTYPE html>
<html lang="es">
<head>
    <!-- Lo que va en <head> NO se ve en la página -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Página</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <!-- Todo lo visible va dentro de <body> -->
    <h1>¡Hola mundo!</h1>
</body>
</html>
💡 Recuerda: <head> es la configuración (no se ve). <body> es el contenido visible.

HTML 2. Etiquetas HTML más usadas

Títulos y texto

<h1>Título principal</h1>      <!-- El más grande, solo 1 por página -->
<h2>Subtítulo</h2>             <!-- Sección -->
<h3>Sub-subtítulo</h3>         <!-- Sub-sección -->
<p>Un párrafo de texto.</p>   <!-- Texto normal -->

Enlaces y botones

<!-- Enlace: te lleva a otra página -->
<a href="https://google.com">Ir a Google</a>

<!-- Enlace a otra página de tu sitio -->
<a href="proyecto1.html">Ver Proyecto 1</a>

<!-- Botón -->
<button>Contacto</button>

Imágenes

<!-- Imagen local (en la misma carpeta) -->
<img src="foto.jpg" alt="Descripción de la foto">

<!-- Imagen de internet -->
<img src="https://example.com/foto.jpg" alt="Foto">
⚡ Tip: Siempre pon el atributo alt="" en las imágenes. Describe qué es la foto. Ayuda a personas con discapacidad visual y a Google.

Contenedores (div)

<!-- div = "caja" para agrupar elementos -->
<div class="padre">
    <div class="hijo1">Contenido 1</div>
    <div class="hijo2">Contenido 2</div>
</div>
🧱 ¿Para qué sirve el div? Por sí solo no hace nada visual. Pero es la pieza clave para organizar tu página y aplicar CSS (especialmente Flexbox).

CSS 3. CSS — Cómo darle estilo

CSS controla cómo se ve la página. Se escribe en un archivo aparte (styles.css) y se conecta con:

<link rel="stylesheet" href="styles.css">

Estructura de una regla CSS

selector {
    propiedad: valor;
    otra-propiedad: otro-valor;
}

Propiedades más comunes

body {
    background-color: #f0f0f0;    /* Color de fondo */
    font-family: Arial, sans-serif; /* Tipografía */
    margin: 0;                     /* Quitar márgenes por defecto */
    color: #333;                   /* Color del texto */
}

h1 {
    font-size: 24px;               /* Tamaño del texto */
    color: #2c3e50;               /* Color del título */
    text-align: center;            /* Centrar texto */
}

img {
    width: 200px;                  /* Ancho fijo */
    height: 200px;                 /* Alto fijo */
    border-radius: 100%;           /* Círculo perfecto */
}

button {
    background-color: #3498db;    /* Fondo del botón */
    color: white;                  /* Texto blanco */
    padding: 10px 20px;            /* Espacio interno */
    border: none;                  /* Sin borde */
    border-radius: 8px;            /* Bordes redondeados */
    cursor: pointer;               /* Manito al pasar mouse */
}

CSS 4. Selectores CSS

Los selectores le dicen al CSS a qué elemento aplicar el estilo:

/* Por etiqueta — afecta a TODOS los h1 */
h1 { color: blue; }

/* Por clase — afecta solo a los que tengan class="padre" */
.padre { display: flex; }

/* Hijo dentro de padre — img que está DENTRO de .hijo1 */
.hijo1 img { width: 200px; }

/* Por ID — afecta solo al que tenga id="titulo" (único) */
#titulo { font-size: 32px; }
⚡ Regla de oro: Usa clases (.nombre) para todo. Los IDs (#nombre) son para cosas únicas. Las etiquetas (h1, p) para estilos globales.

CSS 5. El Box Model

Cada elemento en HTML es una caja con 4 capas:

margin (espacio afuera)
border (borde)
padding (espacio adentro)
Contenido 
.caja {
    margin: 20px;       /* Espacio AFUERA de la caja */
    border: 2px solid black; /* Borde visible */
    padding: 15px;      /* Espacio ADENTRO de la caja */
}

/* Atajo: margin y padding con 4 valores */
.caja {
    margin: 10px 20px 10px 20px;  /* arriba derecha abajo izquierda */
    padding: 10px 20px;           /* arriba-abajo  derecha-izquierda */
}
❌ Error clásico: No entender por qué hay "espacio de más" entre elementos. Casi siempre es por el margin o padding por defecto. Usa margin: 0 en el body para empezar limpio.

Layout 6. Flexbox — Alinear cosas

Flexbox es la forma más fácil de alinear elementos. Se activa en el padre:

.padre {
    display: flex;              /* ¡Activa Flexbox! */
    justify-content: center;   /* Centrar horizontalmente */
    align-items: center;       /* Centrar verticalmente */
}

flex-direction: ¿fila o columna?

row (por defecto) →

📐 Resumen rápido:
justify-content → distribuye en el eje principal (horizontal por defecto)
align-items → alinea en el eje cruzado (vertical por defecto)

HTML CSS 7. Ejemplo: Página de Currículum

Esto es lo que construimos hoy en clase. Un CV con foto, nombre y proyectos:

HTML (index.html)

<!DOCTYPE html>
<html lang="es">
<head>
    <link rel="stylesheet" href="styles.css">
    <meta charset="UTF-8">
    <title>Mi Currículum</title>
</head>
<body>
    <!-- Sección superior: foto + nombre -->
    <div class="padre">
        <div class="hijo1">
            <img src="foto.jpg" alt="Mi foto">
        </div>
        <div class="hijo2">
            <h1>Nombre: Juan Pérez</h1>
            <h2>Descripción:</h2>
            <button>Contacto</button>
        </div>
    </div>

    <!-- Sección de proyectos -->
    <div class="proyectos">
        <div class="proy1">
            <h1><a href="proyecto1.html">Proyecto 1</a></h1>
        </div>
        <div class="proy2">
            <h1><a href="proyecto2.html">Proyecto 2</a></h1>
        </div>
    </div>
</body>
</html>

CSS (styles.css)

/* Estilos globales */
body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
    margin: 0;
}

/* Header: foto + info lado a lado */
.padre {
    display: flex;               /* Pone hijos en fila */
    justify-content: center;    /* Centra horizontalmente */
    align-items: center;        /* Centra verticalmente */
}

/* Foto circular */
.hijo1 img {
    width: 200px;
    height: 200px;
    border-radius: 100%;        /* Círculo */
    margin-right: 20px;         /* Espacio entre foto y texto */
}

/* Proyectos en fila, distribuidos */
.proyectos {
    display: flex;
    flex-direction: row;
    justify-content: space-evenly; /* Espacio igual entre cada uno */
    align-items: center;
}
