1. Repaso: 会 (huì) — Yo sé / puedo hacer

✅ Afirmativo: Sujeto + 会 + Verbo + Objeto
❌ Negativo: Sujeto + 不会 + Verbo + Objeto
❓ Pregunta: Sujeto + 会 + Verbo + Objeto + 吗？
Respuesta: 会 (sí, sé) / 不会 (no, no sé)
🤏 Modesto/tierno: Sujeto + 会 + 一点儿 (yìdiǎnr) → Sé un poquito

🤏 会一点儿 — Sé un poquito (forma modesta/tierna)

🔑 Agregar 一点儿 (yìdiǎnr) después de 会 suaviza la respuesta. Es más humilde y tierno, como decir "sé un poquito nada más".
HanziPinyinEspañol
我会一点儿。Wǒ huì yìdiǎnr.Sé un poquito.
我会说一点儿汉语。Wǒ huì shuō yìdiǎnr Hànyǔ.Sé hablar un poquito de chino.
我会写一点儿汉字。Wǒ huì xiě yìdiǎnr Hànzì.Sé escribir un poquito de caracteres.
我会做一点儿中国菜。Wǒ huì zuò yìdiǎnr Zhōngguó cài.Sé cocinar un poquito de comida china.
我会画一点儿画。Wǒ huì huà yìdiǎnr huà.Sé pintar/dibujar un poquito.

✅ Ejemplos afirmativo / negativo

HanziPinyinEspañol
我会说汉语。Wǒ huì shuō Hànyǔ.Yo sé hablar chino.
我会写汉字。Wǒ huì xiě Hànzì.Yo sé escribir caracteres.
我会做饭。Wǒ huì zuòfàn.Yo sé cocinar.
我会游泳。Wǒ huì yóuyǒng.Yo sé nadar.
我会开车。Wǒ huì kāichē.Yo sé manejar.
我会唱歌。Wǒ huì chànggē.Yo sé cantar.
我不会做中国菜。Wǒ bú huì zuò Zhōngguó cài.No sé cocinar comida china.

🎨 画画 (huàhuà) — Pintar / Dibujar

🔑 画 (huà) tiene doble función:
• Como verbo: pintar, dibujar → 我会画。 (Sé pintar.)
• Como sustantivo: pintura, dibujo → 这幅画很美。 (Esta pintura es muy bella.)

画画 (huàhuà) = verbo + objeto → literalmente "pintar pinturas". Forma completa y natural.
HanziPinyinEspañol
我会画画。Wǒ huì huàhuà.Yo sé pintar/dibujar.
我不会画画。Wǒ bú huì huàhuà.No sé pintar/dibujar.
她很会画画。Tā hěn huì huàhuà.Ella pinta/dibuja muy bien.

💃 跳舞 (tiàowǔ) — Bailar

🔑 Mismo patrón verbo + sustantivo:
• 跳 (tiào) = saltar · 舞 (wǔ) = baile
• 跳舞 (tiàowǔ) = literalmente "saltar baile" → bailar
HanziPinyinEspañol
我会跳舞。Wǒ huì tiàowǔ.Yo sé bailar.
我不会跳舞。Wǒ bú huì tiàowǔ.No sé bailar.
她很会跳舞。Tā hěn huì tiàowǔ.Ella baila muy bien.
💡 Patrón verbo + sustantivo: Muchos verbos en chino funcionan así:
• 唱歌 (chànggē) = cantar (唱) + canción (歌)
• 跳舞 (tiàowǔ) = saltar (跳) + baile (舞)
• 画画 (huàhuà) = pintar (画) + pintura (画)
• 做饭 (zuòfàn) = hacer (做) + comida (饭)

2. Repaso: 很 (hěn) + Adjetivo — 好 vs 难

✅ Afirmativo: Sujeto + 很 + Adjetivo → Él/Ella es muy ___
❌ Negativo: Sujeto + 不 + Adjetivo → Él/Ella no es ___

Describir personas

HanziPinyinEspañol
她很矮。Tā hěn ǎi.Ella es muy baja.
他很高。Tā hěn gāo.Él es muy alto.
她很瘦。Tā hěn shòu.Ella es muy delgada.
他很胖。Tā hěn pàng.Él es muy gordo.
她很漂亮。Tā hěn piàoliang.Ella es muy bonita.
他很帅。Tā hěn shuài.Él es muy guapo.
他很聪明。Tā hěn cōngming.Él es muy inteligente.
他不高。Tā bù gāo.Él no es alto.
她不丑。Tā bù chǒu.Ella no es fea.

👁️ 好看 (hǎokàn) — Bonito (para cosas)

好看 = 好 (bueno) + 看 (ver) → "bueno de ver" → bonito.
Según la profesora: se usa más para cosas, películas, libros, ropa, no tanto para personas.
HanziPinyinEspañol
这个电影很好看。Zhège diànyǐng hěn hǎokàn.Esta película es muy buena.
这本书很好看。Zhè běn shū hěn hǎokàn.Este libro es muy bueno.
你的衣服很好看。Nǐ de yīfu hěn hǎokàn.Tu ropa es muy bonita.
这个不好看。Zhège bù hǎokàn.Esto no es bonito.

🍜 好吃 (hǎochī) — Delicioso

好吃 = 好 (bueno) + 吃 (comer) → "bueno de comer" → delicioso.
HanziPinyinEspañol
中国菜很好吃。Zhōngguó cài hěn hǎochī.La comida china es muy deliciosa.
智利菜很好吃！Zhìlì cài hěn hǎochī!¡La comida chilena es muy deliciosa!
炒饭很好吃。Chǎofàn hěn hǎochī.El arroz frito es muy delicioso.
火锅很好吃。Huǒguō hěn hǎochī.El hot pot es muy delicioso.
包子好吃吗？Bāozi hǎochī ma?¿Los baozi son deliciosos?
中国菜不好吃。Zhōngguó cài bù hǎochī.La comida china no es deliciosa.

🤢 好 vs 难 — Bueno vs Malo

难 (nán) = difícil / malo → es el opuesto de 好 cuando se combina con verbos.
好 (hǎo) — Bueno难 (nán) — Malo
好吃 (hǎochī) — delicioso难吃 (nánchī) — asqueroso
好看 (hǎokàn) — bonito难看 (nánkàn) — feo
好喝 (hǎohē) — sabroso (bebida)难喝 (nánhē) — asqueroso (bebida)
好听 (hǎotīng) — suena bien难听 (nántīng) — suena mal
HanziPinyinEspañol
这个菜很难吃。Zhège cài hěn nánchī.Este plato es muy feo/asqueroso.
这个电影很难看。Zhège diànyǐng hěn nánkàn.Esta película es muy mala.
这个歌很难听。Zhège gē hěn nántīng.Esta canción suena muy mal.

3. 怎么 (zěnme) — ¿Cómo?

Concepto importante: 怎么 (zěnme) siempre va antes de un verbo. Pregunta la manera o el método de hacer algo.
Estructura: Sujeto / Objeto + 怎么 + Verbo ？
⚠️ 怎么 va antes del verbo, nunca después:
✅ 这个字怎么写？ (Correcto)
❌ 这个字写怎么？ (Incorrecto)

怎么 + diferentes verbos

怎么 + VerboPinyinEspañol
怎么写zěnme xiě¿Cómo se escribe?
怎么读zěnme dú¿Cómo se lee/pronuncia?
怎么说zěnme shuō¿Cómo se dice?
怎么做zěnme zuò¿Cómo se hace?
怎么去zěnme qù¿Cómo se va? / ¿Cómo llego?
怎么吃zěnme chī¿Cómo se come (esto)?
怎么用zěnme yòng¿Cómo se usa?
怎么学zěnme xué¿Cómo se aprende?

💡 Para preguntar el nombre es más natural usar:
你叫什么名字？ (Nǐ jiào shénme míngzi?) — ¿Cómo te llamas?
你的名字怎么说？ es más como "¿cómo se pronuncia tu nombre?"

5. Diálogos de Práctica

Conversación 1: ¿Sabes escribir caracteres?

A: 你会写汉字吗？
Nǐ huì xiě Hànzì ma? — ¿Sabes escribir caracteres chinos?
B: 我会写。
Wǒ huì xiě. — Sé escribir.

Conversación 2: ¿Cómo se escribe este carácter?

A: 这个字怎么写？
Zhège zì zěnme xiě? — ¿Cómo se escribe este carácter?
B: 对不起，这个字我会读，不会写。
Duìbuqǐ, zhège zì wǒ huì dú, bú huì xiě. — Lo siento, este carácter sé leerlo, pero no sé escribirlo.
